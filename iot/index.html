Overview

Investigate the components that make up an Internet of Things (IoT) computing environment. In this module we examine the fundamental design issues required by IoT, including foundation distributed and mobile systems. In particular we investigate those current and emerging hardware and software technologies that are used to support a range of IoT applications. The module also includes a number of case studies where existing and emerging technologies are examined.

Requisites Statement

Students must have a basic understanding of communication networking environments, both from a hardware and software perspective. Previous programming knowledge using Java must be equivalent to that taught in computer algorithms and data structures. The prerequisite knowledge would have been covered in the level 4 Introduction to Programming (INTPROG) module.

Learning Outcomes

  1. Critically evaluate the applicability of IoT technologies, application areas and their associated development environments.
  2. Critically evaluate the current and emerging issues in research and the development of IoT, encompassing current architectures, technologies, applications and trends.
  3. Develop effective applications to exploit commercially available and popular modules
