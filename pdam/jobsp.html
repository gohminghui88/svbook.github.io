Some of you may asked about what is the jobs prospect of computer science. For general information, you may refer to:
https://www.prospects.ac.uk/careers-advice/what-can-i-do-with-my-degree/computer-science

From my perspective, you can proceed to developer path or non-developer path.

Developer Path or Software Engineer:

For developer path, usually, you will need to slowly progress towards enterprise Java (Enterprise Java Beans, Struts, MVC, ...) from your java course, or enterprise .Net (ASP.Net MVC) from C#, depending on your location and technology advancements. Java, Python, C# are still the base programming language that usually will be there. In development, you can take up:

  • - front-end developer (UI design, HTML, CSS, JS) - the beautiful website you sees everyday
  • - back-end developer (EJB, Struts, ...) - business logic: how the website react to the web forms you filled up
  • - database related roles (SQL, MS SQL, ...) - where the form data filled up by you stored
  • - Mobile App Developer (Java for Android, Objective C for iPhone, C# for windows Apps)
  • - Software Tester

    (industrial cert to accompany in years to come: MCSD, oracle certified enterprise architect (OCEA) )

For Non-Developer Path, you can focus on network, cyber security, and etc.
  • - Network Administrator (industrial cert to accompany: CCNA, CCNP, MCSE)
  • - Business Analyst
  • - Security Professional (industrial cert to accompany: CISSP, Certified Ethical Hacker, AISP)

For Multi-Disciplinary:
  • - Data Science Analyst or Data Scientist (Need Statistics, Computer Science (with Artificial Intelligence and data mining), preferably some business knowledge)
  • - Internet of Things, IoT (Need Computer Science and some hardware related Engineering)

Hopefully, you can plan your path from here. For myself, I am in the Data Science field.
